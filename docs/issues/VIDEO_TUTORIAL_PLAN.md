## Video Overview

**Purpose:** Create an engaging "How It Works" video tutorial that demonstrates OneLink's core features and value proposition to potential users visiting the landing page.

**Target Audience:** 
- First-time visitors to the landing page
- Users considering signing up
- People who want to understand OneLink quickly

## Content Planning

### What to Show?

#### Option 1: Full User Journey (Recommended - First-Time User)
1. **Introduction (0:00 - 0:10)**
   - Quick intro: "OneLink is your all-in-one link-in-bio platform"
   - Jump straight to sign-in page (no landing page needed - video is on landing page)

2. **Sign Up Flow (0:10 - 0:40)**
   - Show sign-in page with Google OAuth option
   - Click "Continue with Google" (show OAuth flow briefly)
   - Welcome page - choosing username (highlight this key feature)
   - Dashboard first view

3. **Creating Links (0:35 - 1:05)**
   - Add a new link (e.g., Instagram, Twitter, YouTube)
   - Show drag-and-drop reordering
   - Save and see it on profile

4. **Creating Drops (1:05 - 1:35)**
   - Create a new drop (e.g., "Resume" or "Portfolio")
   - Upload a file
   - Show share link generation
   - Demonstrate public/private toggle

5. **Profile View (1:35 - 2:05)**
   - Show public profile with links and drops
   - Demonstrate clicking a link
   - Show drop file upload interface

6. **Analytics & Settings (2:05 - 2:20)**
   - Quick glimpse of analytics dashboard
   - Settings overview
   - Call to action: "Get started for free"

#### Option 2: Feature-Focused (Alternative)
- Focus on 3-4 key features only
- More detailed demonstration of each
- Better for specific use cases
