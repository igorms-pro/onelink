## üì¶ WHAT (The Product)

**OneLink is a unified link-in-bio platform that combines routing and file sharing into a single branded profile.**

### Product Components:

1. **Public Profile Page** (`/:slug` or custom domain)
   - Clean, modern design
   - Customizable branding
   - Mobile-responsive
   - Fast and reliable

2. **Routes (OneMeet)**
   - Intent-first buttons linking to external destinations
   - Support for schedulers (Calendly, Cal.com, Koalendar)
   - Links to documentation, portfolios, or any URL
   - Click tracking and analytics

3. **Drops (DropRequest)**
   - Public file inboxes
   - No login required for visitors
   - Optional metadata collection (name, email, notes)
   - File upload with size limits per plan
   - Owner views submissions in dashboard inbox

4. **Dashboard**
   - Profile management
   - Route and Drop creation/editing
   - Submission inbox (for Drops)
   - Analytics and click tracking
   - Custom domain configuration
   - Billing and plan management

5. **Plans & Features**
   - **Free**: 1 profile, 3 actions, 50MB/file, 7-day retention
   - **Starter ($5/mo)**: Unlimited actions, 1GB/file, 30-day retention, custom domain
   - **Pro ($10/mo)**: 5GB/file, 90-day retention, advanced fields, GA integration

### Technical Specifications:

- **Frontend**: React (Vite), Tailwind CSS
- **Backend**: Supabase (Auth, Postgres, Storage)
- **Billing**: Stripe (Checkout + Portal)
- **Hosting**: Vercel with custom domain support
- **Monitoring**: Sentry, PostHog
